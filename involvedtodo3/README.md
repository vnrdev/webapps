# Todo App

This is a simple AngularJS Todo app.

![demo](Screenshots/demo.png)

### Stack
- AngularJS
- Entity Framework
- WebApi
- Ninject

### Patterns
- Repository Pattern
- Inversion of Control
- Dependency Injection

### Requirements
- AngularJS 1.4.8
- Bootstrap 3.3.6
- Angular Resource 1.2.13
- SQL Server Express 2012 LocalDB
